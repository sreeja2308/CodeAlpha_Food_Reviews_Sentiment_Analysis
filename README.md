# CodeAlpha_Food_Reviews_Sentiment_Analysis
Sentiment Analysis project on Food Reviews dataset using Python, TextBlob, Pandas, and Matplotlib to classify customer reviews as positive, negative, and neutral.
# Food Reviews Sentiment Analysis

This project performs Sentiment Analysis on customer food reviews using Natural Language Processing (NLP) techniques.

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib
- Google Colab

## Features
- Positive Review Detection
- Negative Review Detection
- Neutral Review Detection
- Sentiment Visualization

## Dataset
Amazon Fine Food Reviews Dataset from Kaggle.
